Geek went to a stationery shop with an amount of N in his hand. There are 4 types of pens, each of which cost Rs 1, Rs 2, Rs 5, Rs 10. Geek wants to buy a minimum number of pens, as he has limited space in his pocket but, at the same time he wants to spend all his money. The task is to find the minimum number of pens geek will buy.

Note: Shop has an infinite number of pens of each type

Input:
1. The first line of the input contains a single integer T denoting the number of test cases. The description of T test cases follows.
2. The first line of each test case contains a single integers N.


Output: For each test case, print the answer

Constraints:
1. 1 <= T <= 105
2. 1 <= N <= 109

Example:
Input:
2
6
10
 

Output:
2
1

Explanation:
Test Case 1: Buy a pen of cost Rs. 5 and Rs. 1
